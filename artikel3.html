Bootstrap adalah framework front-end open-source yang digunakan untuk membangun website modern dengan cepat. Framework ini berisi kumpulan CSS, JavaScript, dan komponen siap pakai untuk membuat tampilan website lebih rapi, interaktif, dan responsif di berbagai perangkat. Bootstrap pertama kali dikembangkan oleh tim Twitter pada tahun 2011, sehingga dulu sering disebut Twitter Bootstrap. Kenapa Menggunakan Bootstrap? 1. Responsif Otomatis – Tampilan website menyesuaikan layar HP, tablet, atau laptop. 2. Cepat & Mudah – Banyak komponen siap pakai (button, card, navbar, modal). 3. Desain Konsisten – Gaya tampilan seragam tanpa banyak coding CSS manual. 4. Banyak Digunakan – Dokumentasi lengkap dan komunitas besar. 5. Mudah Dikustomisasi – Bisa diubah sesuai kebutuhan dengan CSS tambahan.
